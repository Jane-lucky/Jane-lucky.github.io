hexo初始化基础配置文件解析

_config.yml

网站的配置信息，可以配置大部分参数。

站点设置

参数 描述
title 网站标题
subtitle 网站副标题
description 网站描述
keywords 网站关键词
author 作者名称
language 使用语言
timezone 网站时区

网址设置

package.json

应用程序信息

scaffolds

模板文件夹，当新建文章的时候会根据该模板建立文件。默认是post模板(可以自己添加模板内容)

source

资源文件夹是用来存放用户资源的地方。

HTML和MarkDown文件会被解析并放到public文件夹，而其他文件会被拷贝过去

themes

主题文件夹。根据该主题来生成静态页面。
